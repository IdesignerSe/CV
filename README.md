# 📌 Two‑Column HTML CV

A clean, modern two‑column CV layout built with pure **HTML** and **CSS**.  
Designed for simplicity, readability, and easy customization.  
Includes a left sidebar for personal details and a right column for professional content.

---

## 🚀 Features

- Two‑column responsive layout  
- Round profile photo section  
- Skills with star‑rating visualization  
- Clean typography and spacing  
- Easy to customize colors, fonts, and sections  
- No frameworks, no JavaScript — fully lightweight

---
